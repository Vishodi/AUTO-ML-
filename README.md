## Getting Started

1. **Clone this repository:**
    ```
    git clone https://github.com/Piyush-xo-19/AUTO-ML-.git
    cd Gothgpt
    ```

2. **Install dependencies:**
    ```
    pip install -r requirements.txt
    ```

3. **Run locally:**
    ```
    streamlit run app.py
    ```
